== Description ==

WB Carousel is an easy plugin to add carousel in WordPress sites.This plugin is responsive and  works using shortcode .

>[Plugin Demo](http://carousel.wpbrim.com) | [Documentation](http://docs.wpbrim.com/wb-carousel-docs/)|[Support](https://wpbrim.com/support/)

<strong> Plugin Features: </strong >

* Easy installable
* Fully responsive OWL Carousel.
* Supported all Modern browsers
* Powered by OWL Carousel 2.
* OWL Carousel Shortcode Ready.
* Custom post type with category.
* Less coding so super weight.
* Mobile, Tablet touch supported.
* Work with all WordPress theme.
* Developer friendly & easy to customize.
* Multiple Carousel can be shown from different Carousel categories.


<strong> Basic Shortcode - in Free version </strong>

<h4> Show All images in a carousel </h4>

<pre>[wb-carousel] </pre>

<h4> Show All images of any category in a carousel in ASC Order </h4>
 
<pre>  [wb-carousel  carousel_cat="pic" order="ASC"]</pre>

 <h4> Show All images of any category in a carousel in DESC Order </h4>
 
<pre>[wb-carousel  carousel_cat="pic" order="DESC"]</pre>



== Installation ==

Installing  WB Carousel is easy like  a regular WordPress plugin. Here goes the step by step Guide to Install WB Carousel plugin :

= Install Plugin using WordPress Admin Panel Search Option =
- In your Admin, go to menu Plugins > Add New
- Search WB Carousel
- Click to install
- Activate the plugin
- Owl Carousel post type will show below POSTS with its owl icon

= Install Plugin using Upload Method in WordPress Admin Panel =

- Download the latest version of WB Carousel (.zip file)
- In your Admin, go to menu Plugins > Add New
- click on the Upload Tab
- Click on “Choose File from your Hard Disk”
- Browse and Select downloaded wb-carousel.zip & click on ‘Install Now’ button
- Activate the plugin
- Owl Carousel post type will show below POSTS with its owl icon

= Install  Plugin using FTP Manager =
- Upload owl-carousel-wp folder with all files inside the /wp-content/plugins/ directory
- Go to WordPress dashboard > Plugins & Activate the  WB Carousel plugin
- Owl Carousel post type will show below POSTS with its owl icon.
